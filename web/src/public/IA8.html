Pivoting:
I could see a situation in the future where we would pivot based on customer discovery/need as we may find that customers want additional features which we can add by updating the software or potentially making changes to the actual hardware model. We have also performed a platform pivot which involved creating a the android app version of our webapp. Another example was the value capture pivot we decided to make based on charging users a per month instead of per year subscription fee for cloud services.
